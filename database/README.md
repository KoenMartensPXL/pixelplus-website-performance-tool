# Database

This folder contains the MySQL schema for the Pixelplus Website Performance Tool.

## Setup

Run `schema.sql` in MySQL Workbench to create the full database from scratch.

Database name:
- `pixelplus_analytics`

## Main tables

- `customers`
- `job_runs`
- `magic_link_tokens`
- `ga4_daily_metrics`
- `ga4_daily_breakdowns`
- `gsc_daily_metrics`
- `gsc_daily_queries`
- `monthly_reports`
